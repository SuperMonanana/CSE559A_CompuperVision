How I make the final 3D models:
  1. Draw 3 lines for xline, yline, zline respectively.
  2. Calculate VPs
  3. Manually enter the values of the 4 lower points, and push them into the stack.
  4. Manually enter the value of 1 upper point and mark as height ref
  5. Compute Homography
  6. Manually assign coordinates to some of the polygons, and then use Same Z Plane and Same XY Plane to calculate the rest, and adjust the unappropiate points.
  7. Create polygons out of each side of the box and name them.
  8. Export as the wrl file type
  9. Convert .tga files to .gif
  10. Import the mesh (.wrl) file into Meshlab
